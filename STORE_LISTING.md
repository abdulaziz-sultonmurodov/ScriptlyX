## Detailed Description (Copy this to Chrome Web Store)

ScriptlyX - Instant Translation & Transliteration

Transform text between languages and scripts effortlessly! ScriptlyX is your all-in-one tool for translating and converting text between English, Russian, and Uzbek (both Latin and Cyrillic scripts).

KEY FEATURES

★ Multi-Language Translation
• English ↔ Russian
• English ↔ Uzbek
• Russian ↔ Uzbek
• Automatic language detection for quick translations

★ Script Transliteration
• Russian: Latin ↔ Cyrillic (phonetic conversion)
• Uzbek: Latin ↔ Cyrillic (official alphabet mapping)
• Instant, offline conversion - no internet required!

★ Simple & Fast
• Right-click any selected text to translate or transliterate
• Text is replaced in-place - no copy-paste needed
• Works in text fields, input boxes, and editable content

★ Quick Translate
• One-click EN↔RU translation with automatic detection
• Cyrillic text? Translates to English
• Latin text? Translates to Russian

★ Privacy-Focused
• Transliteration works 100% offline
• No data collection or tracking
• No account required

HOW TO USE

1. Select any text on a webpage
2. Right-click to open the context menu
3. Choose ScriptlyX → Translate or Transliterate
4. Done! Text is instantly converted

You can also click the extension icon to:
• Test translations in the popup
• Convert text between scripts
• View character mapping tables
• Enable/disable the extension

## Additional Store Information

### Single Purpose Description
ScriptlyX translates and transliterates text between English, Russian, and Uzbek languages, allowing users to convert selected text on any webpage via the right-click context menu.

### Permission Justifications

**contextMenus**: Required to add translation and transliteration options to the browser's right-click context menu.

**activeTab**: Required to access the currently active tab when the user invokes a translation or transliteration action.

**scripting**: Required to inject the content script that replaces selected text with translated/transliterated text.

**storage**: Required to save user preferences such as enabled/disabled state and notification settings.

### Remote Code Policy
This extension does not use any remotely hosted code. All translation API calls are made directly to third-party translation services (Google Translate, MyMemory) and only the translated text is received and used.
